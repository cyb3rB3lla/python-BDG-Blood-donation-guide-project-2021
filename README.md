# python-BDG-Blood-donation-guide-project-2021
BDG is a simple Python program designed to guide blood donors by checking eligibility and showing the nearest hospital that accepts donations. Using lists, loops, and a basic GUI, the system makes the donation process easier and aims to expand to more cities in the future.
